# Inworld TTS Chrome Extension

Chrome extension that converts text to speech using the Inworld TTS API. Includes a popup for quick generation and a settings page for full configuration.

## Features
- Popup text input with language and voice selection
- Settings page for API key, model, encoding, sample rate, and streaming
- Uses selected text from the active page
- Audio playback in the popup

## Setup
1. Open Chrome and go to the Extensions page.
2. Enable Developer mode.
3. Click "Load unpacked" and select this project folder.

## Usage
1. Click the extension icon.
2. Enter text or use the selected text button.
3. Choose language and voice.
4. Click Generate Speech.

## Settings
Open the Settings button in the popup to configure API key and advanced options.
